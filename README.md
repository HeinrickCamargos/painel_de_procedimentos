📂 Sistema Interno de Procedimentos (POP Digital)
Este é um sistema de gerenciamento de Procedimentos Operacionais Padrão (POP) desenvolvido para organizar o conhecimento técnico e operacional de uma empresa em "Blocos" lógicos.

🚀 Principais Funcionalidades:
Organização por Blocos: Agrupamento de tarefas por categorias ou departamentos (ex: TI, Financeiro, FAQ).

Editor Rich Text (Quill.js): Registro de procedimentos detalhados com suporte a formatação de texto e colagem direta de imagens (prints).

Gestão de Setores: Atribuição de setores responsáveis para cada procedimento individual.

Anexos: Upload e vinculação de documentos auxiliares (PDF, Word, Imagens).

Busca Inteligente: Localização rápida de procedimentos ou blocos através de uma barra de pesquisa global.

Interface Moderna: Layout responsivo, focado na experiência do usuário e na clareza das informações.

🛠️ Tecnologias Utilizadas:
Backend: Python com Flask.

Banco de Dados: SQLAlchemy (SQLite).

Frontend: Jinja2, Bootstrap 5 e Quill.js.
